Cocktail Concluder

Components
  • React
  • HTML + CSS + Javascript
  • TheCocktailDB
  • Python API requests
  • React API requests

A react app that allows you to save money by showing how common different cocktail ingredients are. Putting in a combination of ingredients shows you not only the drinks you can make but which single ingredient gives you the most new options. Created to help students build out their collection in a cost effective manner. Uses data obtained from TheCocktailDB with a python script and manages all the data statically¹ inside the site itself and hosted on Netlify.


1: Static data storage was chosen due to the tiny size of the data and the speed at which not calling a database improves speed and cellular data usage. The whole site is 4% the size of the facebook landing page and has a Total Blocking Time of 0ms on average.
