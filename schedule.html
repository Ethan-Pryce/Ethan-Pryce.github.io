Schedulebot.net

Components
  • React
  • Node.js
  • Express
  • MongoDB (NoSQl)
  • CSS
  • Google Accounts

Built using the MERN stack Schedulebot.net is a service for helping find the optimal time to host events. It allows a user to invite other users to input the favoribility of times while also allowing the user to set the priority for each user. It updates every three seconds from the database to emulate concurrent calendar editing. Hosted on Heroku and Netlify.
