Schedule-bot.com

Components
  • React
  • Node.js
  • Express
  • MongoDB (NoSQl)
  • CSS + HTML + Javascript
  • Google Accounts

Built using the MERN stack, Schedule-bot.com is a service for helping find the optimal time to host events. It allows a user to invite other users to input the favoribility of times while also allowing the user to set the priority for each user. It then calculates the best time to host the event and reports it. The site is designed to help organize a group with varying schedules when some times are prefered, originally focused specifically on helping students get togther. Hosted on Vercel and Netlify.
